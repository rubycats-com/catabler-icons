# Catabler::Icons
https://github.com/tabler/tabler-icons icon pack

## Installation
Add this line to your application's Gemfile:

```ruby
gem "catabler-icons"
```
Use in erb template

```erb
<%= render IconComponent.new(:cat) %>
<%= render IconComponent.new(:paw) %>
```
